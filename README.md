# Welcome to React TanStack Ideas

## You will find in this repo the following stuff

A simple CRUD application built with **React**, **TanStack Router**, **TanStack Query**, **TypeScript**.

This project is designed to:

- Explore file-based routing with **TanStack Router**.
- Learn server state management using **TanStack Query**.
- Practice CRUD operations with a Mock API.
- Understand route loaders and data prefetching.
- Build a modern React application using TypeScript.
- Improve component organization and project structure.

---

# Tech Stack

- React
- TypeScript
- Vite
- TanStack Router
- TanStack Query
- Axios
- JSON Server (Mock REST API)

---

# Features

- View all ideas
- View idea details
- Create a new idea
- Update an existing idea
- Delete an idea
- Route data prefetching
- Automatic cache management
- Loading and pending states
- Type-safe routing

# Getting Started

## Clone repository

```bash
git clone https://github.com/gahoccode215/idea-drop-ui
```

## Install dependencies

```bash
npm install
```

## Start JSON Server

```bash
npx json-server db.json --port 8000
```

## Run the project

```bash
npm run dev
```
